### Prerequisites
- Python 3.12 or later installed on your machine
- Basic familiarity with the command line/terminal

### Installation

1. **Clone the repository**
   ```bash
   
   git clone https://github.com/Th3-Beast/BeginnerProjects.git
   
   ```

